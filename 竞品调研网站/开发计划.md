## 开发原则

1. 不为了完成任务而修改代码
2. 需求不明确先提问再开发
3. 优先保证代码可维护性，而不是速度
4. 每次开发前先阅读本文档，不依赖聊天记录
5. 发现可优化的地方提出建议，不未经确认直接修改

## 技术栈

- 后端：Python 3.12 + FastAPI + SQLAlchemy 2.0 + PostgreSQL
- 前端：Next.js 15 + TypeScript + Tailwind CSS
- AI：OpenAI GPT-4o (Structured Outputs)
- 搜索：Tavily API
- 定时：APScheduler
- 认证：JWT (passlib + python-jose)
- 部署：Docker Compose
