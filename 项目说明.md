# AI 实验室 · 项目说明

> 这是个人 AI 学习和项目工作的统一管理仓库。

## 仓库用途

这个仓库用于管理我的 AI 学习项目和工作项目，方便后续持续使用和回顾。

## 目录结构说明

```
AI实验室/
├── 项目说明.md          ← 你正在看的这个文件
├── 学习路线.md          ← AI 学习路径规划
├── 更新记录.md          ← 整体更新日志
├── Prompt库/            ← 收集和整理的提示词
├── 学习笔记/            ← 学习过程中的笔记
├── UI设计/              ← 通用 UI 设计稿和灵感
├── AI学习Demo/          ← 学习用的小型 Demo 项目
├── 第二大脑/            ← 知识管理系统
├── 竞品调研网站/        ← 第一个正式项目（开发中）
└── 实验项目/            ← 各种实验性项目
```

## 当前项目

### 竞品调研网站（进行中）
一个"知→行"信息差消除平台，输入关键词自动聚合信息、AI 精炼成时间线、输出行动建议。
- 技术栈：FastAPI + Next.js + PostgreSQL + OpenAI + Tavily
- 详细文档见 `竞品调研网站/` 目录

## 使用规范

1. 所有文档默认中文
2. 文件夹使用中文命名
3. 代码目录保留英文（src、public、node_modules 等）
4. 代码注释使用中文
5. 每次开发结束自动生成开发总结
6. 自动更新 `更新记录.md`
7. 每次开始开发前先阅读相关项目文档
