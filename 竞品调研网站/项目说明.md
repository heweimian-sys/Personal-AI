# 竞品调研网站 · 项目说明

> 本文档是项目的唯一事实来源，每次开发前请先阅读本文档。

## 一、项目是什么

一个"知→行"信息差消除平台。

你输入任意关键词（比如"AI 行业趋势""深圳经济政策""某竞品动态"），网站会：
1. 自动搜索互联网上的相关信息
2. 用 AI 把信息整理成时间线（按日期排列的事件）
3. 分类标记（产品、政策、财务、技术等）
4. 生成结构化报告和行动建议

**一句话总结**：输入关键词，了解过去和现在发生了什么，获得行动建议。

## 二、为谁做

- **个人**：了解某个领域/行业的历史和现状（市场、政策变化）
- **企业**：一份自动生成的竞品调研报告，了解竞品在过去的周期做了什么

## 三、三种使用方式

| 方式 | 说明 | 类比 |
|---|---|---|
| 搜索式 | 输入关键词，立刻获得报告 | 类似搜索引擎 |
| 订阅式 | 订阅主题，系统定期自动追踪更新 | 类似 Google Alerts |
| 对话式 | 和 AI 对话，深入分析某个主题 | 类似 ChatGPT |

## 四、技术栈（选用的工具）

| 层面 | 技术 | 为什么选它 |
|---|---|---|
| 后端语言 | Python | 生态好，AI 库丰富 |
| 后端框架 | FastAPI | 速度快，自动生成 API 文档 |
| 数据库 | PostgreSQL | 稳定可靠，适合存储结构化数据 |
| AI 模型 | OpenAI GPT-4o | 能力强，支持结构化输出 |
| 搜索引擎 | Tavily API | 专为 AI 优化，返回清洗后的内容 |
| 前端框架 | Next.js (React) | 主流，生态成熟 |
| 定时任务 | APScheduler | 轻量，不需要额外服务器 |
| 认证 | JWT | 简单可靠 |
| 部署 | Docker | 一键启动所有服务 |

## 五、设计风格

水墨文学风：宣纸白底 + 墨色文字 + 青瓷绿点缀。
字体用宋体（中文）+ Cormorant Garamond（英文/数字），营造书卷气。
