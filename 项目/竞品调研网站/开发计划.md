# 竞品调研网站 · 开发计划

> 本文档是开发执行的依据，每次开发前请阅读当前进度，从断点继续。
> 本版为 v3 MVP 版本，共 16 个任务。

## 开发原则

1. 不为了完成任务而修改代码
2. 需求不明确先提问再开发
3. 优先保证代码可维护性，而不是速度
4. 每次开发前先阅读本文档，不依赖聊天记录
5. 发现可优化的地方提出建议，不未经确认直接修改

## 技术栈

- 后端：Python 3.12 + FastAPI + SQLAlchemy 2.0 + PostgreSQL
- AI：DeepSeek API（OpenAI SDK 兼容，JSON Output 模式）
- 搜索：Firecrawl 自部署（完全免费，爬取+结构化）
- 前端：Next.js 15 + TypeScript + Tailwind CSS
- 认证：单一访问密码（非多用户体系）
- 测试：pytest + httpx，核心链路覆盖
- 参考：deep-research（github.com/dzhng/deep-research）、deep-research-web-ui
