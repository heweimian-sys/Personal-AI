# 新人网站部署 SOP

这是面向新人使用的静态网站部署 SOP，覆盖从 GitHub 到 Cloudflare Pages 的前端上线流程，以及 Cloudflare D1 + Pages Functions 的轻量后端配置流程。

## 线上入口

- 网站：<https://website-deploy-sop.pages.dev/>
- 后端 D1 页面：<https://website-deploy-sop.pages.dev/backend.html>
- 飞书文档：<https://shengcaiyoushu01.feishu.cn/docx/CIVAdh1ViopZOBxwmiAc2M8pnhf>

## 本目录内容

- `index.html`：新人网站部署 SOP 主页面
- `backend.html`：Cloudflare D1 后端部署页面
- `assets/`：网站使用的脱敏、压缩后图片资源
- `docs/飞书文档.md`：飞书文档入口和维护说明

## 安全说明

GitHub 仓库只保存适合公开展示的网站图片。飞书文档中使用的是未脱敏原图，仅用于内部文档查阅，不提交到 GitHub。

## 维护方式

后续更新网站内容时，先更新本目录的 HTML 和 assets，再提交到 GitHub。Cloudflare Pages 绑定仓库后会自动重新部署。
