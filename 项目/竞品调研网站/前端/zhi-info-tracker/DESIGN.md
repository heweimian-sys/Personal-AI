# 知信 (ZhiXin) 设计系统说明

## 项目概述

知信是一个一站式信息调研平台，包含 5 个页面：
首页、登录、工作台、调研报告、知识图谱。

---

## 一、当前设计体系

### 1. 颜色

| 用途 | 色值 | CSS 变量 |
|------|------|----------|
| 主品牌色（蓝色） | `#009afc` | `--blue` / `--color-surf-blue` |
| 主色 hover | `#006dc8` | `--blue-dark` / `--color-deep-current` |
| 背景色 | `#f3f4f6` | `--bg` / `--color-mist` |
| 卡片/表面 | `#ffffff` | `--card` / `--color-paper` |
| 正文文字 | `#000000` | `--ink` |
| 辅助文字 | `#5b6882` | `--ink-secondary` / `--color-slate-blue` |
| 弱提示文字 | `#a0a0a0` | `--ink-muted` |
| 边框 | `#e5e7eb` | `--border` / `--color-hairline` |
| 强边框 | `#d1d5db` | `--border-strong` |
| 成功 | `#10b981` | - |
| 警告 | `#f59e0b` | - |
| 错误 | `#ef4444` | - |

### 2. 字体

- **UI / 正文**：系统无衬线（`-apple-system`, PingFang SC, Noto Sans SC, Microsoft YaHei）
- **展示/标题**：衬线（Georgia, Noto Serif SC, SimSun）
- 行高：正文 1.5，标题紧凑

### 3. 圆角

- 按钮/输入框：`10px`（`--radius`）
- 卡片：`14px`（`--radius-lg`）
- 标签/徽章：`9999px`（全圆角）

### 4. 阴影

- 卡片静态：`0 1px 2px rgba(0,0,0,0.04)`
- 卡片 hover：`0 4px 12px rgba(0,0,0,0.06)`
- 搜索框聚焦：`0 0 0 3px var(--blue-light)`

### 5. 间距

- 页面最大宽度：`1200px`
- 页面内边距：`24px`
- 栅格间距：`20px`
- 区块间距：`80px` padding top/bottom

### 6. 常用组件

| 组件 | 特征 |
|------|------|
| 导航栏 | 固定顶部，滚动后有毛玻璃背景，无衬线 logo |
| 按钮(.btn-primary) | 蓝色填充，10px 圆角，`_ :hover → 深蓝` |
| 按钮(.btn-outline) | 白色底 + 灰边，`hover → 蓝边蓝字` |
| 搜索栏 | 白色底 + 灰边，聚焦时蓝色发光边框，右侧蓝色按钮 |
| 模式切换器 | 三个选项"全网 | 资讯 | 学术"，无容器，用竖线分隔 |
| 状态标签(badge) | 蓝色浅底 + 圆点动画 + 文字 |
| 功能卡片 | 白色 + 灰边 + 微阴影 + 14px 圆角，hover 阴影增强 |
| 标签(pill) | 蓝色浅底 + 蓝色文字 + 全圆角 |
| 幽灵链接 | 文字链接，hover 变蓝色 + 下划线动画 |
| 呼吸小圆点 | 6px 蓝色圆点，`pulse` 动画循环 |

### 7. 动画

- `pulse`：圆点呼吸缩放（4s 循环）
- `fadeUp`：元素入场向上淡入（0.6s）
- 过渡：hover 状态 0.2s

### 8. 页面结构

| 页面 | 布局 | 特点 |
|------|------|------|
| 首页 | 全屏 Hero + 三列功能卡片 + Footer | 蓝天辉光背景 |
| 登录 | 居中卡片 + 社交登录 | 蓝天辉光 + 呼吸圆点 |
| 工作台 | 左侧固定导航 + 右侧内容区 | 统计卡片 + 滚动更新列表 |
| 报告 | 左侧导航 + 中间阅读区 + 右侧目录 | 蓝色左侧色块标记段落 |
| 知识图谱 | 左侧导航 + 全屏 SVG 图 | 结构化知识节点 |

## 二、当前风格分析

### "严肃"的来源

1. **颜色**：蓝色 + 灰色系，偏理性冷静。没有活泼的颜色（如橙色、粉色、紫色）
2. **字体**：系统默认无衬线 + 老式衬线，没有圆润/可爱的字体
3. **圆角**：10-14px 适中，不是超大圆角
4. **阴影**：非常浅，几乎无立体感
5. **装饰元素**：很少（只有背景辉光和呼吸圆点）
6. **动效**：只有淡入和呼吸动画，没有弹跳、弹性、重力感

### 如果想增加"游戏感/趣味性"，可以改的方向

- 增加活泼的强调色
- 加大圆角（20px+）
- 增加阴影深度和投影颜色
- 加入弹跳动画、弹性动画
- 使用更圆润的字体
- 加入 emoji、图标动效、骨架屏
- 卡片增加倾斜/重叠排列

## 三、跟 AI 沟通 UI 改动的"通用语言"

> 下面这个模板可以直接复制粘贴给任何 AI（Claude、GPT 等），用结构化的方式说出你的需求：

### 标准 prompt 模版

```
## 目标
[一句话说明你要做什么]

## 当前设计参考
- 参考风格：[具体网站/设计系统名字]
- 参考链接：[URL]
- 参考页面：[截图]

## 设计调性
我想让页面感觉更 [活泼/有趣/轻松/游戏感/可爱/专业/...]
具体来说：
- 色彩：想要 [更鲜艳/更柔和/撞色/渐变/...]
- 字体：想要 [更圆润/更艺术/更有力量感/...]
- 圆角：想要 [更大/更小/不统一]
- 阴影：想要 [更深/更浅/彩色阴影]
- 动效：想要 [弹跳/弹性/漂浮/抖动/...]

## 不动部分
- 布局结构不变
- 页面数量不变
- 功能逻辑不变

## 改动的页面
- pages/index.html
- pages/dashboard.html
- [其他页面...]

## 交付要求
- 所有样式写在单个 HTML 文件中，不要外部依赖
- 提供完整的 HTML 文件内容
```

### 简写版（快速沟通）

```
风格参考：[链接/截图]
想要的感觉：[活泼/可爱/游戏感/潮牌/科技感/...]
具体改动：
1. 颜色 → [想要的配色描述]
2. 圆角 → [大小]
3. 字体 → [想要的字体风格]
4. 动效 → [想要的动画类型]
5. 其他 → ...
不改变：布局、功能、页面数量
```

### 如果你的需求是"参考某张图/某个网站"

```
请参考这个设计风格：[粘贴截图/链接]
提取它的核心设计特征（颜色、字体、圆角、阴影、动效），
应用到我的网站上。保持布局不变。
```

## 四、AI 设计对话常用词汇表

| 中文描述 | AI 理解的关键词 | 对应 CSS |
|---------|----------------|---------|
| 更活泼 | playful, vibrant | 亮色 + 弹性动画 + 大圆角 |
| 游戏感 | game-like, gamified | 霓虹色 + 发光阴影 + 弹跳动画 |
| 更柔和 | soft, gentle | 降低对比度 + 更浅阴影 + 大圆角 |
| 科技感 | tech, futuristic | 深色底 + 发光边框 + 渐变 |
| 潮牌 | streetwear, brand | 撞色 + 粗字体 + 大字号 |
| 可爱 | cute, kawaii | 粉色系 + 超大圆角 + 圆润字体 |
| 极简 | minimal, clean | 大量留白 + 无装饰 + 小字号 |
| 轻量 | light, airy | 浅阴影 + 小圆角 + 克制色彩 |
| 有质感 | premium, refined | 精致阴影 + 细腻渐变 + 好字体 |
| 温暖 | warm, cozy | 暖色（橙/黄/红）+ 圆角 + 柔和阴影 |
| 弹跳动画 | bounce animation | `animation: bounce 0.5s` |
| 弹性动画 | spring animation | `cubic-bezier(0.68, -0.55, 0.27, 1.55)` |
| 漂浮动画 | float animation | `translateY` 循环变化 |
