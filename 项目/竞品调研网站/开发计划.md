# 知信 · 认知加速器 — 开发计划

> 本文档是开发执行的依据，每次开发前请阅读当前进度，从断点继续。
> v5 版本：4 周 4 阶段

## 开发原则

1. 不为了完成任务而修改代码
2. 需求不明确先提问再开发
3. 优先保证代码可维护性，而不是速度
4. 每次开发前先阅读本文档，不依赖聊天记录
5. 发现可优化的地方提出建议，不未经确认直接修改

## 技术栈

- 后端：Python 3.12 + FastAPI + SQLite
- AI：DeepSeek API（OpenAI SDK 兼容，JSON Output）
- 搜索：Firecrawl 自部署（完全免费，爬取+结构化）
- 前端：Next.js 15 + Tailwind CSS + Inter / Playfair Display
- 部署：Phase 1 先本地跑通
- 参考：deep-research、deep-research-web-ui
